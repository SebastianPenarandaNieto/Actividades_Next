# Simpsons Explorer

Aplicación hecha con **Next.js** y **React** para explorar personajes de *Los Simpson* usando la API pública **The Simpsons API**.

## Funcionalidades
- Búsqueda de personajes por nombre 
- Paginación (1–60) 
- Tarjetas con foto y nombre 
- Modal con detalles completos 
- Cerrar modal con botón o clic fuera 
- Estilos con TailwindCSS 

## Estructura
- /app/page.js                → Lógica principal 
- /components/Character.jsx   → Tarjeta y modal de personaje

## Instalación
```bash
npm install
npm run dev
```
Abrir: `http://localhost:3000`

## API utilizada
https://thesimpsonsapi.com/api/characters?page=1

## Tecnologías
- Next.js 
- React 
- TailwindCSS
